## 🚀 Quick start

- Install

  ```bash
  bun install
  ```

- Run all challenges

  ```bash
  bun test
  ```

- Run specific challenge

  ```bash
  bun test YYYY/challenge-XX # where YYYY is the year and XX is the challenge number
  ```
